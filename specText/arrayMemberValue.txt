arrayMemberValue
bool arrayMemberValue(int a[], int value, unsigned ind, unsigned length)
The function returns true if there exists an element that equals value in the subarray a[ind...length - 1]
Input: a = {1,3,122,2,2}, value = 122, ind = 1, length = 5. Output: true
Input: a = {0,90,1,2,2}, value = 0, ind = 2, length = 5. Output: false
