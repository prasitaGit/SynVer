checkListArraySame
bool checkListArraySame(struct sll *h, int a[], unsigned index, unsigned length)
The function retuens true if all elements of the subarray a[index...length - 1] and list l are same.
Input: h -> {3->122->2->2}, a = {1,3,122,2,2}, index = 1, length = 5. Output: true
Input: h -> {2->0}, a = {1,3,122,2,2}, index = 3, length = 5. Output: false
