isOddAtIndexOdd
bool isOddAtIndexOdd(int a[], unsigned ind, unsigned length)
The function returns true if the elements at odd indices are also odd in the subarray a[ind...length - 1]
Input: a = {1,3,2,2,2}, ind = 1, length = 5. Output: false
Input: a = {1,3,2,7,2}, ind = 0, length = 5. Output: true
