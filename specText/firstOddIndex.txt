firstOddIndex
int firstOddIndex(int a[], int ind, int length)
The function returns the index of the first element that is odd in the subarray a[ind...length - 1]. If no such index exists, it returns -1.
Input: a = {1,3,2,2,2}, ind = 1, length = 5. Output: 1
Input: a = {1,3,2,2,2}, ind = 2, length = 5. Output: -1
