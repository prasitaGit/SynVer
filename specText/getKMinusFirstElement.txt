getKMinusFirstElement
int getKMinusFirstElement(int a[], int k, int length)
The function returns the element at (k - 1)st index if k - 1 is within array bounds. It returns -1 if (k - 1) is out of bounds
Input: a = {1,3,2,2,2}, k = 5, length = 5. Output: 2
Input: a = {1,3,1,4,5}, k = 0, length = 5. Output: -1
