checkNGreaterthanAll
bool checkNGreaterthanAll(int a[], int n, unsigned ind, unsigned length)
The function returns true if n is greater than all elements in the subarray a[ind...length - 1]
Input: a = {1,3,122,2,2}, n = 122, ind = 1, length = 5. Output: false
Input: a = {0,90,1,2,2}, n = 3, ind = 2, length = 5. Output: true
