containsConsecutiveNumbers
bool containsConsecutiveNumbers(int a[], unsigned ind, unsigned length)
The function returns true if there exists two consecutive indices i and i + 1, where a[i] + 1 = a[i + 1] for the subarray a[ind...length - 1], and false otherwise
Input: a = {1,3,2,2,2}, ind = 1, length = 5. Output: false
Input: a = {1,3,1,2,2}, ind = 2, length = 5. Output: true
