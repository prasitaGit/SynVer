allNumsSame
bool allNumsSame(int a[], unsigned ind, unsigned length)
The function returns true if all elements are same for the subarray a[ind...length - 1]
Input: a = {1,3,2,2,2}, ind = 1, length = 5. Output: false
Input: a = {1,3,2,2,2}, ind = 2, length = 5. Output: true
