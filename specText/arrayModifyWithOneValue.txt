arrayModifyWithOneValue
void arrayModifyWithOneValue(int a[], int value, unsigned index, unsigned length)
The function updates all values of the subarray a[index...length - 1] to value
Input: a = {1,3,122,2,2}, value = 122, index = 1, length = 5. Output: {1,122,122,122,122}
Input: a = {0,90,1,2,2}, value = 0, ind = 2, length = 5. Output: {0,90,0,0,0}
