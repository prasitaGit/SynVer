lastElementPosition
int lastElementPosition(int a[], int value, int ind, int length)
The function returns the last index at which value is found in the subarray a[ind...length - 1]. If no such index exists, it returns -1.
Input: a = {1,3,2,2,2}, value = 2, ind = 1, length = 5. Output: 4
Input: a = {1,3,2,2,2}, value = 3, ind = 2, length = 5. Output: -1
