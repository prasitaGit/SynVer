addAllElementsInRangeBy1
void addAllElementsInRangeBy1(int a[], unsigned ind, unsigned length)
The function increments all elements of the subarray a[ind...length - 1] by 1
Input: a = {1,3,122,2,2}, ind = 1, length = 5. Output: {1,4,123,3,3}
Input: a = {0,90,1,2,2}, ind = 2, length = 5. Output: {0,90,2,3,3}
