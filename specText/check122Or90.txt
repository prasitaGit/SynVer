bool check122Or90(int a[], unsigned ind, unsigned length)
The function returns true if there exists an element that equals 122 or 90 in the subarray a[ind...length - 1]
Input: a = {1,3,122,2,2}, ind = 1, length = 5. Output: true
Input: a = {1,90,1,2,2}, ind = 2, length = 5. Output: false
